Fidelity export instructions:
  1. Log into fidelity.com → Accounts & Trade → Positions
  2. Click Download (top right) → save the CSV as your Portfolio file
  3. Go to Accounts & Trade → History → set date range → Download
  4. Save that CSV as your History file
Schwab export instructions:
  1. Log into schwab.com → Accounts tab → Positions
  2. Click Export (top right) → save CSV as your Portfolio file
  3. Go to Accounts → History → select All Transactions and date range
  4. Click Export → select CSV → save as your History file
IBKR export instructions:
  1. Log into IBKR Portal → Performance & Reports → Flex Queries
  2. Create a Trade Confirmation Flex Query with all option fields, export as CSV for History
  3. Go to Portfolio → Positions → use the export/download option for Portfolio
